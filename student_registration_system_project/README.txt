----PROJECT DESCRIPTION----

I created a student registration system where we will only keep the student's name and surname data.

*A student can be added to the list by creating a record by entering the student name and surname.
*The record can be deleted from the list by entering the student name and surname.
*The user can create as many student records as he wants.
*The user can delete as many student records as he wants.
*user can view the students in the list with their numbers. (The student's index number on the list is considered as the student number)
